    https://velog.io/@polynomeer/동적-계획법Dynamic-Programming
    https://www.leafcats.com/71
    https://www.leafcats.com/72
    겹치는 부분 문제 : 어떤 문제가 여러 개의 부분 문제로 쪼개질 수 있을때 재귀 알고리즘으로 해결
    최적 부분 구조 : 어떤 문제의 최적의 해결책이 그 부분 문제의 최적의 해결책으로 부터 설계될 수 있는 경우
    메모제이션 : 동적계획법에서는 문제를 한 번만 풀어야 한다. 정답을 한번 구하면 캐시에 메모해 놓는다. 시간 복잡도 O(N)
    구현 방식
        Top-down : 큰 문제를 작은 문제로 쪼갠다. 재귀. 피보나치
        Bottom-up : 작은 문제부터 차례대로 푼다. 반복
