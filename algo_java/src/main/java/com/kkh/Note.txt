    겹치는 부분 문제 : 어떤 문제가 여러 개의 부분 문제로 쪼개질 수 있을때 재귀 알고리즘으로 해결
    최적 부분 구조 : 어떤 문제의 최적의 해결책이 그 부분 문제의 최적의 해결책으로 부터 설계될 수 있는 경우
    메모제이션 : 동적계획법에서는 문제를 한 번만 풀어야 한다. 정답을 한번 구하면 캐시에 메모해 놓는다. 시간 복잡도 O(N)
    구현 방식
        Top-down : 큰 문제를 작은 문제로 쪼갠다. 재귀. 피보나치
        Bottom-up : 작은 문제부터 차례대로 푼다. 반복

    프림 알고리즘(그리디 알고리즘)
        한 노드를 기준으로 전체를 생각하지 않고 현재 상황에서 최선의 선택만 고려
        임의의 노드를 출발 노드로 선택한다.
        출발 노드를 포함하는 트리를 점점 확장해간다.
        매 단계에서 이미 트리에 포함된 노드와 포함되지 않은 노드를 연결하는 에지들 중 가장 가중치가 작은 에지를 선택한다.
    크루스칼 알고리즘(그리디 알고리즘)
        가장 작은 간선부터 선택해가면서 순환이 생기지 않게 선택
        간선들을 가중치의 오름차순으로 정렬한다.
        간선들을 그 순서대로 하나씩 선택하되, 이미 선택된 간선들과 사이클이 형성되면 선택하지 않는다.
        N - 1 개의 간선이 선택되면 종료한다.
